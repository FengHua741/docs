# 简介

    * 一些驱动注意事项


## 驱动安装

    * 请注意需要EN对准左上角安装
    * FLY主板驱动口的EN脚都在左上方
    * FLY的DP5、PRO X10、MMU只能使用TP驱动
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.png').default} size="80%" align="center" />

## 驱动配置

    * TP驱动和普通驱动的配置方法是一样的
    * FLY-2240驱动支持UART连接，但是需要手动修改电阻
    * 驱动采样电阻需要注意，如果配置错误可能会导致电机、驱动损坏
