## 参考配置

    ```cfg
    [fan]
    ##    Print Cooling Fan 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## 开机自动转解决方法

    :::info
    * 这边只简单说明
    * 在固件编译时候添加上PWM的信号引脚，并且反转他状态
    * 测试使用SUPER8PRO，引脚使用PF8,在固件编译时候选择`GPIO pins to set at micro-controller startup`添入`!PF8`即可
    * 请注意填入的`!`是英文的感叹号
    * 其他主板同理
    :::
