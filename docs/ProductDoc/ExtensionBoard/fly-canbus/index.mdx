# CANBUS

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-canbus/img/canbus.png').default} size="40%" align="left" />


## 产品特点

    * 支持CAN与电源一分四，可一次性扩展四个CAN工具板
    * 板载RJ11接口，设计用于与使用CAN协议的系统进行连接和通信。虽然RJ11在CAN系统中并非标准接口，但我们的产品提供了将二者顺畅连接的可能
    * 人性化设计，通过打印件可直接安装到导轨上

## 产品规格

    * 通信接口：RJ11\*1,XH2.54*5
    * 电源接口：KF128-5.028-2P*5
    * 输入电压：5V-24V
    * 兼容性：  RRF/Klipper/等CAN

## 注意事项

    :::warning 注意事项

    * 请注意，电源输入多少就输出多少
    * 请注意，线序不要接错
    * 确保在安装扩展板前，上位机已经关闭电源。
    * 避免在潮湿或灰尘环境中使用扩展板。
    * 安装设备时，请注意工具版的电压规格，以免造成损坏。

    :::
