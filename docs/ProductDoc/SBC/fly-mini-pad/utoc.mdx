# 板载UTOC使用

:::danger 须知

不建议使用两个UTOIC，烧录与连接会非常麻烦

:::
    * 如果不需要使用板载UTOC需要执行下方命令，执行不会有任何提示
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * 还原方法，执行不会有任何提示
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * MINI Pad已经板载了一个UTOC
