# 简介

      Fly_WiScreen是广州镁伦电子科技有限公司Fly3D主板研发团队推出的一款高端Klipper控制屏幕，可通过Wifi控制3D打印机，支持SD卡和U盘传输打印文件，屏幕响应速度非常快给用户非常好的体验。支持设置主从机模式，进行多机同步操作方便用户管理机器。其中5V供电口可接充电模块用于改装电池方便移动。
        
    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen5/img/wiscreen.png').default} size="100%" align="left" />

## 产品功能

    * 支持通过WIFI连接其他安装好Klipper的上位机
    * 支持WIFI搜索，只需要连接WIFI后可自行搜索上位机
    * 支持集群控制多台打印机
    * 支持使用USB与TF卡上传打印
    * 自定义宏，操作调试更方便

## 产品参数

    * CPU: 全志T113-S3,双核 CORTEX- A7最高1.2GHZ
    * 内存: 集成128M DDR3
    * 存储: 128MB Nand Flash，TF卡座
    * 显示: 800X480电容屏
    * 外置接口: 一路USB 2.0，一路TF卡座
    * WIFI协议: 802.11b/g/n
    * 屏幕类型: IPS
    * 屏幕尺寸: 5英寸
    * 多点触控: 支持
    * 额定电压: 5V
