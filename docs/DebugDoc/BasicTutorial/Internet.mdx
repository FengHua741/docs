# 通过SSH连接网络


* 请确保系统已经启动并且通过SSH使用串口连接到上位机，推荐使用**MobaXterm_Personal**
* 请确保上位机已经安装**M2WE**或者**2.4G_WiFi模块**，如果是**Lite2**与**MiniPad**则不需要安装模块
* 请注意**2.4G_WiFi模块**,**Lite2**与**MiniPad**这三个只能使用2.4G_WiFi无法使用5GWiFi
* 请注意网络不能使用双频合一否则无法正常连接
* 请确保天线已经安装完毕！！！

 1. 在SSH软件输入**nmtui**命令后回车，会出现下图所示界面

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.png').default} size="80%" align="center" />

 2. 回车然后选择**Activate a connection**在回车

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.png').default} size="80%" align="center" />

 3. 选择你需要连接的网络，回车输入WiFi密码在**选择OK**进行保存连接

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.png').default} size="80%" align="center" />

 4. 连接好WIFI后，在键盘找打**Esc**按键按几次推出当前界面，然后输入**ip a**并且回车

 * 图中192.168.2.136即为上位机的IP地址

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.png').default} size="80%" align="center" />
