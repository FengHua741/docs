# Function Demonstration

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.png').default} size="100%" align="left" />

* Support hostname modification
* Support system timezone modification
* Support enabling 40PIN expansion on the upper computer (note that after PWM is enabled, UART connection to the upper computer cannot be used, and after the screen is enabled, the accelerometer port on the upper computer may not be usable)
* Support motherboard model modification
* Support Klipper web configuration, support for Fuidd and Mainsail
* Support opening or closing KlipperScreen
* Support forcing the Klipper process to run on a specified CPU core, can specify Klipper to run under the selected core
