:::warning

SD Card Precautions:

* Capacity must be no less than `16GB` or no more than `128GB` for the TF card
* The TF card's transfer speed must be `class10` or higher, it is recommended to use brands like SanDisk

:::

:::danger Notice

* Due to server issues, please do not update the system or klipper arbitrarily
* Please download FLY OS system, installing the original Armbian is too difficult and not recommended

:::

* Please note that the downloaded system needs to be extracted, not just changing the extension or directly burning it!!!!
* If you do not extract and then burn, it will not start!!!!

:::
