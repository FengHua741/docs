# Raspberry Pi Image


  :::warning

  SD card precautions:

  * The capacity must be no less than `16GB` or no more than `128GB` TF card
  * The transmission speed of the TF card must be class10 or higher, it is recommended to use TF cards from brands like SanDisk

  :::

  :::danger Notice

  * Due to server issues, do not update the system or klipper arbitrarily
  * The system image is suitable for Raspberry Pi 3B/3B+/4B/CM4/5B
  * This image is based on `bookworm` installation of various services
  * This image comes with TFT screen enabled by default, if you do not need to use the TFT screen, you need to manually modify `/boot/config.txt`

  :::

  :::info Important Note

  * Please note that the downloaded system needs to be decompressed, not just changing the extension or directly burning the file!!!!
  * Not decompressing and then burning will prevent the system from starting!!!!

  :::
