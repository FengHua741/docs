# {props.name} System Download

:::warning

SD Card Precautions:

* Capacity must be no less than `16GB` or no more than `128GB` for TF card
* The transfer speed of the TF card must be class10 or higher, it is recommended to use brands like SanDisk

:::

:::danger Notice

* Due to server issues, do not update the system or klipper arbitrarily
* Please download FLY OS system, the original Armbian is too difficult to install and is not recommended

:::

:::info Important Notice

* Please note that the downloaded system needs to be decompressed, not just change the extension or directly burn it!!!!
* Burning without decompressing will result in an unbootable system!!!!

:::
