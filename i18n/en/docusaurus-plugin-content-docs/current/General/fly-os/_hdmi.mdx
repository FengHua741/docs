## FLY Host Machine User Guide

:::tip
Open your IP:9999 in the browser to access FLY-Tools.
:::

:::tip
The HDMI interface on the onboard of the FLY host machine is Micro HDMI. Please pay attention when purchasing an HDMI cable!!!
:::

:::tip
The FLY host machine supports FPC One-Line Connection for `FLY-PI-V2`, `FLY-C8`, and `FLY-PRO-X10`.
:::

:::tip
Using an FPC to HDMI module can be compatible with host machines like Raspberry Pi.
:::
