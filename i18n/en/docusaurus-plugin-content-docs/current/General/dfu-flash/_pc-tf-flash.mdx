### Preparation

    * Before flashing the firmware, please first [connect via SSH using the network](/docs/DebugDoc/BasicTutorial/index.mdx "click to jump")
    * Counterfeit memory cards may cause flashing failure, please try to use the recommended memory cards below for firmware update
    * Prepare a TF card with a capacity not exceeding `32G`, and it is recommended that the transfer speed of the TF card must be `class10` or higher, it is suggested to use brands like SanDisk for the TF card

### Download Firmware

    1. Use the prepared TF card
    2. Format using `SDFormatter`
        <ImageView image={require('@site/docs/General/dfu-flash/img/flash1.png').default} size="100%" align="left" />
    3. Use **MobaXterm_Personal** to navigate to the file directory on the left side, find `klipper`, then open `out`
        <ImageView image={require('@site/docs/General/dfu-flash/img/ssh.png').default} size="100%" align="left" />
    4. Download `klipper.bin` to the TF card and rename it to `firmware.bin`
        <ImageView image={require('@site/docs/General/dfu-flash/img/ssh1.png').default} size="100%" align="left" />
    5. Power off the motherboard, insert the TF card into the motherboard
    6. On the motherboard, wait for 10 seconds
    7. Remove the TF card and insert it back into the computer. If the `firmware.bin` in the SD card disappears and `FLY.CUR` appears, the flashing is successful.
        <ImageView image={require('@site/docs/General/dfu-flash/img/flash2.png').default} size="100%" align="left" />
