## Product Introduction

    * Screen Size: 120.7mm*76.0mm
    * Display Area: 108mm*66.2mm
    * Resolution: 800*480
    * Screen Type: IPS
    * Communication Method: DSI/Mipi
    * On-board Interface: 15 pin DSI / 22pin Mipi
    * Working Temperature Range: 0°-60°C
    * Touch Type: Capacitive (Capacitive)
    * Logic Voltage: 3.3V
    * Weight: 135.5g

        :::danger Important Notice

        * The DSI and CSI interfaces on the Raspberry Pi 5 are now called MIPI.
        * The MIPI interface can be configured to directly use a CSI camera or DSI screen.
        * Please pay attention to the direction of the header pins.

        :::

        * The Raspberry Pi 5 now has two MIPI interfaces. When viewed from the front, there are markings 1 and 0.

        * Wiring for DSI Screen

            * If using the 1st interface, configure the screen as follows:

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * If using the 0th interface, configure the screen as follows:

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
