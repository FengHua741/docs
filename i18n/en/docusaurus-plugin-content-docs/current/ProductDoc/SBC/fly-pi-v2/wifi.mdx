# Connect to Wi-Fi


## Install the Antenna on PI V2

    * The red arrow points to the Wi-Fi antenna interface.
    * The Wi-Fi port is `ipex1`.
    * Install the antenna on PI V2, otherwise you may not be able to connect to the internet due to poor signal.
