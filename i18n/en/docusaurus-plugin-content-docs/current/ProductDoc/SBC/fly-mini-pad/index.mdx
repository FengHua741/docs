# Introduction

    Fly-MINI-Pad is an entry-level Klipper host computer

## Features

    * CPU: High-performance Allwinner H3 chip, quad-core 64-bit Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Supports up to 128GB SD card
    * WIFI: Onboard 2.4G WiFi
    * Peripherals: SPI x1, USB x2, Type-C x1
    * Interfaces: Three XH2.54-2P interfaces respectively support POWER-IN power supply supporting up to 24V input, CAN-BUS supports CAN bus connection, LI-PO supports 3.7V and protected LiPo battery power/supply interface
    * Pre-installed system includes Klipper, Moonraker, Mainsail, Fulidd, Klipper-Screen, resonance compensation plugin numpy and FLY-Tools, no need for complex operations like changing sources, making it easier for beginners (requires purchasing TF card)
    * Screen: Capacitive/resistive screen, resolution 320*480
