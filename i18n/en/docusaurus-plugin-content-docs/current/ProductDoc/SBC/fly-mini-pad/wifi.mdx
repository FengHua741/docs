# Connect to WiFi

## Install the Antenna onto the MINI PAD

    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Install the antenna onto the MINI PAD, otherwise poor signal may result in an inability to connect to the internet
