## Preparations Before System Flashing

    * Card Reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, the transmission speed of the TF card must be Class10 or higher
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">System Image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Flashing Tool</Button>

## System Flashing

:::info
TF card flashing and flashing to M2WE via TF card only requires using one of these methods.
:::

:::info
* If the machine has M2WE and memory card installed, the system will boot from the memory card by default
* It is recommended to use SD card for flashing M2WE, as flashing via USB may fail due to driver compatibility issues; it is recommended to directly use [Memory Card Flashing to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note that the downloaded system image should match; otherwise, it will not start
:::
