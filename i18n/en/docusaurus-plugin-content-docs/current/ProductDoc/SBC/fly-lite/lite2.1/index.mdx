# Introduction

    Fly-Pi-lite2.1 is an entry-level Klipper host
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.png').default} size="80%" align="center" />

## Features

    * CPU: High-performance Allwinner H3 chip, quad-core 64-bit Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Supports up to 128GB SD card
    * WIFI: Onboard 2.4G WiFi
    * Peripherals: SPI x1, URAT x2, USB x2, HDMI x1
    * Interfaces:预留FPC，HDMI + USB FPC一线通 (Reserved FPC, HDMI + USB FPC all-in-one)
    * Pre-installed system: Klipper, Moonraker, Mainsail, Fulidd, Klipper-Screen, resonance compensation plugin numpy and FLY-Tools, no need for complex operations like changing sources, making it easier for beginners (requires purchasing an SD card)

## Updated Features

    * TFT V1 interface upgraded to TFT V2
    * Added reset and power-off pins
    * Added power-on indicator light
    * Removed BOOT button, replaced with reset button
    * Optimized wiring
