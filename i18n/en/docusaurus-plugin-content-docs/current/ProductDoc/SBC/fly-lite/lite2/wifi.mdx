# Connect to WIFI

## Install the Antenna on LITE 2

    * The red arrow points to the WIFI antenna interface.
    * The WIFI antenna port is `ipex1`.
    * Install the antenna on LITE 2, otherwise you may not be able to connect to the internet due to poor signal.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.png').default} size="100%" align="left" />  

* If you find that you are not connecting to WIFI normally, first check if the antenna is connected, and then use the method below to connect to WiFi.
