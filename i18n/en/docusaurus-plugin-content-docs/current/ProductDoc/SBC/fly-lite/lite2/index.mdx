# Introduction

    Fly-Pi-lite2 is an entry-level Klipper host

## Features

    * CPU: High-performance Allwinner H3 chip, 4-core 64-bit Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Supports up to 128GB SD card
    * WIFI: Onboard 2.4G WiFi
    * Peripherals: Spi x1, URAT x2, USB x2, HDMI x1
    * Interfaces:预留FPC，HDMI + USB FPC一线通
    * Pre-installed system: Klipper, Moonraker, Mainsail, Fulidd, Klipper-Screen, resonance compensation plugin numpy and FLY-Tools, no need for complex operations like changing sources, making it easier for beginners (requires purchasing TF card)
