## Reference Configuration

    ```cfg
    [fan]
    ##    Print Cooling Fan 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## Solution for Automatic Spin on Startup

    :::info
    * This is a brief explanation
    * Add the PWM signal pin and invert its state during firmware compilation
    * Test using SUPER8PRO, use pin PF8, add `!PF8` to `GPIO pins to set at micro-controller startup` during firmware compilation
    * Note that the entered `!` is an English exclamation mark
    * The same applies to other motherboards
    :::
