# Connecting Klipper

    :::info Important Notice

    * The onboard antenna of this product can only be used for receiving, the transmission antenna needs to be installed by yourself, the antenna terminal is `ipex3`
    * Please install the antenna properly!!! Otherwise, poor signal will cause serious heating!!!
    * You need to manually connect to the AirClick hotspot first, then connect to the corresponding WIFI

    :::


    :::info Important Notice

    * AirClick sends commands to the Klipper control machine after connecting to Moonraker
    * Custom macros, emergency stop, Klipper restart, firmware restart, system shutdown, etc., can be achieved

    :::

## Connecting AirClick to WIFI

    * Connect to the AirClick hotspot, the hotspot name is `FLY-WBTN-XXXXXX`
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.png').default} size="50%" align="Center" />
    * After connecting to the hotspot, access [192.168.4.1](http://192.168.4.1/#/setting) through a browser
    * This will display the WIFI settings interface where you need to manually enter the password of the router that your printer is connected to and click the blue box below to connect to WIFI
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.png').default} size="50%" align="Center" />
    * After connecting to WIFI, it will prompt the IP address of the AirClick after connection. Click the blue box below to restart the device
        * Number 1 circled is the IP address of the AirClick
        * Number 2 circled copy the IP and restart the device
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.png').default} size="80%" align="Center" />
    * Please note that you need to manually switch your current network to the corresponding router, otherwise you will not be able to access

## Changing Language

## Connecting to Klipper

    :::info Important Notice
    * AirClick sends commands to the Klipper control machine after connecting to Moonraker
    * Custom macros, emergency stop, Klipper restart, firmware restart, system shutdown, etc., can be achieved
    :::

    * Click on the System Settings on the left
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.png').default} size="80%" align="Center" />
    * Find the `Moonraker IP` below and enter the IP address of the Klipper machine
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-1.png').default} size="80%" align="Center" />
        * Number 1 enter the IP address of the Klipper host
        * Number 2 defaults to `7125`, if the host uses another port, you need to enter the corresponding port manually
        * Number 3 confirm and save the configuration
    * After correct connection, a prompt will appear at the top of the webpage
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.png').default} size="80%" align="Center" />
