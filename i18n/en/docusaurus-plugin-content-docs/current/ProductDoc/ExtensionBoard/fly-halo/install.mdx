## Installation to Voron Stealthburner

    * Halo installation to Stealthburner requires the use of the adapter provided by us.
    * Download the two STL files we provide and print them yourself.
        * [Stealthburner_main_body(Adapt_to_Halo).stl (click to download)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl (click to download)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step (click to download)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### Installation of Adapter Board and Connection Wires
    * The wire exit position can be adjusted according to your preference and does not necessarily have to be against the back of the fan.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.png').default} size="50%" align="left" />

    :::warning Note

    * Wire color explanation: **Black is negative GND**, **Red is positive VCC with an acceptable voltage range of DC12-24V**, **White is GPIO47 which can be used for material detection**, **Yellow is GPIO48 which can be used for machine shutdown after completion**.

    :::
