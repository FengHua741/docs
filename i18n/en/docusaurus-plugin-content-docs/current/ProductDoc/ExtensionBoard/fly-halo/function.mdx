# FLY_halo

## 1. Home Screen

    Click the heated bed or nozzle, slide clockwise from the top of the screen to the desired temperature, and then click the heated bed or nozzle to set the temperature.

    If you need to display MCU temperature and host temperature, add this configuration:

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # Associate with MCU (default)

    [temperature_sensor host]        # Host temperature
    sensor_type: temperature_host     # Associate with host
    ```

    :::warning Note

    * Note that the maximum temperature is read from the configuration settings.

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.png').default} size="50%" align="left" />


## 2. Extrusion

    * Long press the icon to set the hotend temperature, extrusion length, and extrusion speed.

    :::warning Note

    * Halo only supports control of a single hotend.

    :::
