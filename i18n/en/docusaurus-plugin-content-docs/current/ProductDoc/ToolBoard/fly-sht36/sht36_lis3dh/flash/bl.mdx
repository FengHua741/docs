# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SHT36 LIS3DH supports CAN
    * Only CAN firmware at 500K and 1M speed is provided here
    :::

## Entering Programming Mode

    :::tip

    The circled part is the `BOOT button`

    :::

    * With the motherboard completely powered off, short the BOOT button and connect one end of the data cable to the motherboard and the other end to your device.
    * Then release the short on the BOOT button
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/boot.png').default} size="50%" align="center" />

            4. Flashing Firmware

            :::tip

            Only one firmware needs to be flashed.

            :::

            * Below is the command to flash the 500K CAN speed firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_500K.uf2
                ```
                        
                * Below is the command to flash the 1M CAN speed firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_1M.uf2
                ```


            5. If you see the prompt below, flashing is successful.

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            After flashing, you need to power off the motherboard completely.
