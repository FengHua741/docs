# Product Introduction

    FLY-SHT36-LIS3DH is a tool board designed by Guangzhou Mellow Electronics Technology Co., Ltd. for 3D printer extruder heads. Using this tool board, four wires can replace the complex wiring between the mainboard and the print head, optimizing the layout. FLY-SHT36-LIS3DH is suitable for 36-step motors and integrates features such as a CAN transceiver, USB port, TMC2209 driver, LIS3DH accelerometer, two controllable fans, and other functionalities. This tool board is compatible with Klipper firmware.

    * Compact design, using a 6-layer board
    * Supports CAN bus connection, providing more stable data transmission with lower latency and better connectivity
    * On-board TMC2209 driver, using UART mode, suitable for extruders
    * On-board ADXL345 accelerometer
    * Supports PT100 (MAX31865)
    * Replaceable fan MOS design
    * Compatible with BLTOUCH, PL08N, and other leveling sensors, integrated XY limit switches
    * Supports 12-24V voltage input
    * MCU: STM32F072
    * Communication: CANBUS, USB
