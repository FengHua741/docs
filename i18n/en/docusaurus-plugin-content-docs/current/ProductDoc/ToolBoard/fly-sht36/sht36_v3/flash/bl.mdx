# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SHT36_V3 supports CAN and RS232
    * Only CAN firmware at 500K and 1M baud rates is provided here
    :::

## Entering Programming Mode

    :::tip

    The arrow points to the `BOOT button`

    :::

    * With the motherboard completely powered off, hold the BOOT button while connecting one end of the data cable to the motherboard and the other end to your device.

            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * Below is the command to flash the 1M CAN baud rate firmware
