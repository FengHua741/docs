# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SHT36 PRO supports CAN
    * Only CAN firmware is provided here, with speeds of 500K and 1M
    :::

## Entering Programming Mode

    :::tip

    The circled part is the `BOOT button`

    :::

    * With the motherboard completely powered off, short the BOOT button and connect one end of the data cable to the motherboard and the other end to your device.
    * Then release the short on the BOOT button
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.png').default} size="100%" align="center" />

            4. Flashing Firmware

            :::tip

            Only one firmware needs to be flashed.

            :::

            * Below is the command to flash the 500K CAN speed firmware
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Below is the command to flash the 1M CAN speed firmware
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. If the flashing is successful, you will see the following prompt

            :::tip

            After flashing, the motherboard needs to be completely powered off once.
