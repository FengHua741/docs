# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * ERCF  supports CAN and USB
    * Here we provide CAN firmware at 500K and 1M baud rates
    :::

## Enter Programming Mode

    :::tip

    The circled part is the `BOOT button`

    :::

    * To enter programming mode, the mainboard must be completely powered off, press the BOOT button, and then connect the data transfer cable to the mainboard and the host computer.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/boot.png').default} size="100%" align="center" />

            :::tip

            Only one firmware needs to be flashed.

            * Below is the command to flash the 500K CAN baud rate firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * Below is the command to flash the 1M CAN baud rate firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * Below is the command to flash the USB firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```

            5. If the flashing is successful, you will see the following prompt

            :::tip

            After flashing, the mainboard needs to be powered off completely once.
