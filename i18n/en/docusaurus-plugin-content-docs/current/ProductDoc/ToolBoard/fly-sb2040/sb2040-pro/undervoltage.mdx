# Driver Undervoltage Solution

## Method One

    :::danger

    * Replace the power supply
    * Add a power filter
    * Replace motor judgment
    * Connect the motor housing to the GND of the tool plate
    * Add `driver_TPFD: 0` in the extruder driver configuration

    :::

## Method Two
    * Connect to the host via SSH
    * Enter the following command
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * This step may be restored after updating Klipper
    :::
