# Product Introduction

    FLY SB2040 PRO is a tool board developed by Guangzhou Melun Electronic Technology Co., Ltd. for the VORON Stealthburner print head. This tool board allows you to replace the complex wiring between the mainboard and the print head with just four wires, optimizing the layout. SB2040 is suitable for stealthburner and afterburner, integrating CAN transceiver, USB port, TMC2240 driver, ADXL345 accelerometer, three controllable fans, and other functions. This tool board is compatible with Klipper and RRF firmware.

## Product Features

    *预留Boot按钮，用户可通过TypeC进入Boot模式更新固件
    * Support for Klipper firmware
    * Temperature sensor support: On-board MAX31865 chip, compatible with PT100
    * On-board ADXL345 accelerometer
    * Reserved RGB interface, on-board 2240 driver, supports various leveling sensors such as klicky, Voron Tap, PL08n, etc.
    * Supports 12-24V voltage, 15A current input, increased heater port, rated current 10A


## Product Specifications


    * MCU: Raspberry Pi rp2040, Dual-core ARM Cortex-M0+@133MHz
    * Connection interface uses XT30 (2+2) connector, supporting 15A current, peak 30A
    * Firmware: Klipper
    * Input voltage: DC12V-24V
    * Logic voltage: DC3.3V
    * On-board sensors: ADXL345 and optional MAX31865
    * Heater port: Heater (HEAT0)
    * Expansion interfaces: Three controllable fans, RGB*1, limit switches*3, heater tube*1, thermistor*1
    * Fan interfaces: Three controllable fans
    * Motor driver: On-board TMC2240
