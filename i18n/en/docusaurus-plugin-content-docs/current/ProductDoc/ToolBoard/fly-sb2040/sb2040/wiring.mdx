## Power Wiring

:::info
* SB2040 does not support reverse polarity protection. Please check the power positive and negative connections before powering on!!! Otherwise, it will damage the SB2040 tool board!!!
:::

    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/img/wiring/power.png').default} size="40%" align="center" />

    * CAN line color represents the function

    | Color | Function            |
    | :--: | :------------------ |
    | Red  | ***DC 12/24V input*** |
    | Black| ***DC ground (GND)*** |
    | Yellow| ***CAN H***       |
    | White| ***CAN L***       |

## Limit Switch Wiring

    * Limit switches have normally open (NO) and normally closed (NC) types. Generally, in 3D printers, it is recommended to use NC. This way, if there is a problem with the limit switch circuit, the system will report an error in time, avoiding unnecessary crashes and protecting the printer.
    * If it is a VORON model, consider changing the installation position of the limit switch. Install the X limit switch on the print head carriage and the Y limit switch on the A motor mount. Additionally, on the SB2040, it is recommended to add a ^ before the limit switch signal to pull it up. For example:
        ```cfg
        [stepper_x]
        endstop_pin: ^sb2040:PA0  # Add ^ before the signal to pull it up
        ```

### TAP
:::tip
* Voron Tap is not recommended to be connected to **24V**. Some versions using **24V** have a high probability of burning out the Tap sensor. This is not a problem with the Fly product, but a design flaw in the Voron Tap. Please be aware!!!
:::

    * Voron Tap is the latest leveling sensor solution released by the Voron team, featuring high precision, strong stability, and good adaptability. When wiring, make sure the positive and negative poles are not reversed, otherwise it may damage the Tap sensor or the tool board.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/img/wiring/tap.png').default} size="40%" align="center" />
