# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SB2040 supports CAN
    * Only CAN firmware is provided here, with rates of 500K and 1M
    :::

## Enter Programming Mode

    :::tip

    The circled part is the `BOOT button`

    :::

    * With the motherboard completely powered off, press and hold the BOOT button while connecting one end of the data transmission cable to the motherboard and the other end to your device.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/boot.png').default} size="50%" align="center" />

            * Below is the command to flash the 500K CAN rate firmware:
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_500K.uf2
                ```
                       
            * Below is the command to flash the 1M CAN rate firmware:
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_1M.uf2
                ```
