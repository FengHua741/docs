    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/4988.png').default} size="50%" align="center" />

## Basic Parameters

    * Suitable for : Marlin/Klipper/RRF firmware
    * Overheat shutdown protection
    * Ground short circuit protection
    * Load short circuit protection

## Installing the FLY motherboard
    * Please note that EN needs to be aligned with the top left corner during installation
    * FLY's DP5, PRO X10, MMU can only use TP drivers
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.png').default} size="80%" align="center" />

    * Drive current adjustment explanation
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/4988-2.png').default} size="100%" align="center" />
    * The 4988 uses normal jumper cap mode and supports configuring 16 microsteps after connecting four jumpers

## Klipper Reference Configuration
    * The 4988 does not require drive configuration such as `[tmc5160 stepper_x]` or `[tmc2209 stepper_x]`, it is not needed, and configuration will prevent normal use
    * Set `microsteps:` to `16`
