# Product Specifications

    * Equipped with a 32-bit ARM Cortex-M7 series STM32H723ZGT6 main control chip running at 550 MHz
    * Firmware: Marlin 2.0/RepRap/Klipper
    * Driver support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Driver mode support: TMC: UART/SPI
    * 8 high-voltage drive channels, supporting 8 independent motor drives, 5 extruders, and 10 PWM fans
    * Display: Serial touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 high-voltage input port, supporting automatic bed leveling sensors: BLTouch, PL08N, etc., Klicky, Voron Tap, etc.
    * Optional limit switch power supply: 5V/12V/24V
    * Using RepRap firmware, it can use the on-board ESP32 WiFi module, supporting up to 1.6 m/s
    * Unique PWM Fan MOS board design, easy to replace when damaged, reducing customer repair difficulty, two 2-pin fan interfaces supporting 4-pin PWM fans
    * PCB uses 2-ounce copper thickness, capable of handling higher current
    * 6 ADC interfaces, supporting up to six thermistors
    * 8 IO interfaces, usable for leveling and limit switches
    * 5 heater interfaces, supporting up to five heaters
    * 2 PMW interfaces
