# Product Parameters


    * 32-bit ARM Cortex-M4 series at 168 MHz, STM32F407ZGT6 chip
    * Firmware: Marlin 2.0/RepRap/Klipper
    * Driver support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Driver mode support: TMC: UART/SPI
    * 8 high-voltage drive positions, supporting 8 independent motor drives, 5 extruders, and 10 PWM fans
    * Display: serial touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 high-voltage input port, supporting automatic bed leveling sensors: BLTouch, PL08N, etc., Klicky, Voron Tap, etc.
    * Optional limit switch power supply: 5V/12V/24V
    * With RepRap firmware, the onboard ESP32 WiFi module can be used, supporting up to 1.6 m/s
    * Unique PWM Fan MOS board design, allowing direct replacement when damaged, reducing customer repair difficulty, two 2-pin fan connectors supporting 4-pin PWM fans
    * PCB uses 2-ounce copper thickness, capable of supporting higher current
    * 6 ADC interfaces, supporting up to six thermistors
    * 8 I/O interfaces, usable for leveling and limit switches
    * 5 heating element interfaces, supporting up to five heating elements
    * 2 PMW interfaces
