# BL firmware

    :::tip

    * Please note that the `bootloader` has already been pre-flashed at the factory and does not need to be re-flashed.

    :::

## Enter Programming Mode


    * There are two methods to enter programming mode.
    * Method 1: With the motherboard completely powered off, short-circuit the BOOT pin and connect the data transmission cable to both the motherboard and your device.
    * Method 2: Connect the data transmission cable to both the motherboard and your device, then short-circuit the BOOT pin and press the RST button, release the RST button, and finally disconnect the BOOT jumper.
