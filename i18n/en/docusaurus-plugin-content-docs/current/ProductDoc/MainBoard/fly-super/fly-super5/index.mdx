# Product Parameters

    * Features a 32-bit ARM Cortex-M7 series STM32H723ZGT6 main control chip with a frequency of 550 MHz
    * Firmware: Marlin 2.0/Reprap/Klipper
    * Driver support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Driver mode support: TMC: UART/SPI
    * 5 high-voltage drive positions, supporting 5 independent motor drives
    * Display: Serial touch screen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 high-voltage IN port, supporting auto bed leveling sensors: BLTouch, PL08N, etc., Klicky, Voron Tap, etc.
    * Optional limit switch power supply: 5V/12V/24V
    * With RepRap firmware, it can use the on-board ESP32 WiFi module, supporting speeds up to 1.6 m/s
    * Unique PWM Fan MOS board design, easy to replace and reduce customer repair difficulty, two 2-pin fan ports can support 4-pin PWM fans
    * PCB uses 2OZ copper thickness, capable of handling higher currents
    * 3 ADC interfaces, supporting up to three thermistors
    * 7 IO interfaces, usable for leveling and limit switches
    * 3 heating element interfaces, supporting up to three heating elements
