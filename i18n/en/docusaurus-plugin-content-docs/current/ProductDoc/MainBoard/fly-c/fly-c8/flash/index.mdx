# Firmware Burning Instructions
    

## How to Determine Which Firmware You Need

    :::tip

    * Note that you only need to flash one of these firmwares.

    :::
    
    * C8 can be connected to the host machine using the following methods
    * <Button variant="contained" disableElevation href="./usb">USB Firmware</Button>
        * Pros: Reliable and stable communication
        * Cons: Data transmission may be affected if multiple USB devices are used
    * <Button variant="contained" disableElevation href="./can">USB Bridge Firmware</Button>
        * Pros: Can connect multiple devices without occupying a USB port
        * Cons: May cause timeout errors when used with `KlipperScreen`.
