# USB Bridge CAN Firmware Flashing

## Start Compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Press the `Q` key, and a **Save configuration** prompt appears, then press the `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * You should now have saved the configuration and exited to the command line interface
    * Enter `make -j4` to start compiling, it will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * If the following content is output at the end, the compilation was successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Firmware Flashing

:::tip
Execute the following command to flash the firmware
:::

    ```bash
    sudo fly-flash -d c8 -h -f ~/klipper/out/klipper.bin
    ```
