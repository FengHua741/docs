# System Imaging

## Guide for New Host Machine Users

    <Button variant="contained" disableElevation href="/docs/DebugDoc/baodian/host/">Guide for New Host Machine Users</Button>

## Preparations Before System Imaging

    * Card Reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, the transfer speed of the TF card must be Class10 or higher
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">system image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">imaging tool</Button>

## System Imaging

:::info
TF card imaging and imaging to M2WE via TF card only requires using one of them
:::

:::info
* If the machine is installed with M2WE and memory card, the system will boot from the memory card by default
* It is recommended to use SD card for system imaging on M2WE. Imaging through USB to M2W may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Imaging to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note whether the downloaded system image is correct. If it is not correct, the system cannot start
:::
