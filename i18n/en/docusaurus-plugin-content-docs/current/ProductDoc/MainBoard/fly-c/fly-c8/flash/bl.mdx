# BL firmware

    :::tip

    * Please note that the `bootloader` has already been pre-flashed at the factory and does not need to be re-flashed.

    :::

## Enter programming mode


    * There are two methods to enter programming mode.
    * First method: With the motherboard completely powered off, hold the BOOT button and connect the data transmission cable to the motherboard and your device, suitable for WINDOWS.
    * Second method: After connecting the data transmission cable to the motherboard and your device, hold the BOOT button and input the command, suitable for the host machine.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.png').default} size="100%" align="center" />

## Flash firmware
    <Tabs>
        <TabItem value="win" label="Windows flashing tutorial" default>

            * The bootloader firmware for C8: <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/firmware/BL/GeminiV3.bin">Click to automatically download the firmware</Button>


            * Set DIP switches `3`, `4` to the right to connect the lower-level machine to the upper-level machine via the built-in USB.
            * Set DIP switches `1`, `2` to the left, otherwise it will affect the connection between the lower-level machine and the upper-level machine.
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.png').default} size="100%" align="center" />
            <WIN />

        </TabItem>
        <TabItem value="linux" label="Host machine flashing tutorial">

            * Set DIP switches `1`, `2` to the right to connect the lower-level machine to the upper-level machine via the built-in USB.
            * Set DIP switches `3`, `4` to the left, otherwise it will affect the connection between the lower-level machine and the upper-level machine.
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.png').default} size="100%" align="center" />
            * After setting the DIP switches as described above, the USB port below will be occupied by the lower-level machine, so do not connect any other devices.
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="50%" align="center" />

            * Ensure that the host machine can access the internet normally.


                ```bash
                sudo wget https://cdn.mellow.klipper.cn/BL/HIDBL.bin -O ~/HIDBL.bin && sudo fly-flash -d c8 -u -f ~/HIDBL.bin
                ```
            * When the prompt below appears, hold down the lower-level machine's BOOT button and press enter.

                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.png').default} size="80%" align="center" />

            * When the image below appears, it indicates that the flashing is successful and you can release the BOOT button.
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/hid3.png').default} size="80%" align="center" />


        </TabItem>
    </Tabs>
