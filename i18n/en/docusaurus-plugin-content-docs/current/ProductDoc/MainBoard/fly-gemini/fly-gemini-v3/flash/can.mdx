# USB Bridge CAN Firmware Flashing

## Start Compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Press `Q` key, then **Save configuration** appears, press `Y` key again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Now the configuration should be saved and you should exit to the command line interface
    * Enter `make -j4` to start compiling, it takes a while
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * The compilation is successful if the following output is shown

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Firmware Flashing

:::tip
Execute the following commands to flash the firmware
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
