# Connect to WIFI


## Install the Antenna on Gemini

    * The red arrow points to the WIFI antenna interface.
    * The WIFI antenna port is `ipex1`.
    * Install the antenna on Gemini, otherwise you may not be able to connect to the internet due to poor signal.
