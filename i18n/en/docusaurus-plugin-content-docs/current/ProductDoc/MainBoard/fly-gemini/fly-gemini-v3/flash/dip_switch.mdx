# DIP Switch Introduction

:::tip
FLY Gemini connects to the host machine via DIP switches; if the positions are incorrect, it will not function properly.
:::

:::danger Note
Please ensure that the DIP switch is set to the specified position to connect properly to the host machine.
:::

## Connecting to Onboard Host Machine

- Set DIP switches `1`, `2` to the right to connect the lower-level device to the host machine via the built-in USB.
- Set DIP switches `3`, `4` to the left, otherwise it will affect the connection between the lower-level device and the host machine.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.png').default} size="50%" align="center" />
- After setting the DIP switches as described above, the lower-level device will occupy the USB port below. Please do not connect other devices.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="50%" align="center" />

## Connecting to External Host Machine

- Set DIP switches `3`, `4` to the right to connect the lower-level device to the host machine via the built-in USB.
- Set DIP switches `1`, `2` to the left, otherwise it will affect the connection between the lower-level device and the host machine.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.png').default} size="50%" align="center" />
