## Product Features

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Supports CAN bridge
    * Onboard 4x2209
    * Onboard DIAG function pins, which enable infinite position home support with simple jumper cap plug and play

## Product Specifications

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Firmware: Klipper
    * Supports 12-24V DC power supply, more stable power supply, eliminating the need for a voltage drop module
    * Driver working modes supported: UART, STEP/DIR
    * Expansion interfaces: BLTouch (Servos, Probe), Proximity switch X1
    * Fans: Two controllable fans X2, Always-on fan X1
    * Temperature sensors: 2x NTC 100K
    * Heating interfaces: 2x controllable ports supporting 10A
