# Onboard Peripheral Introduction

    * When the DIP switch is set to the right, it will enable the USB port next to the Type-C port.
    * When the DIP switch is set to the left, it will enable the RS232 port next to the Type-C port.

## USB Port Introduction

    * It can be used to connect external cameras and other devices to the motherboard.
