## Product Specifications

    * 32-bit ARM Cortex-M4 series 168 MHz, STM32F407ZGT6 chip
    * Supports CAN bus connection, more stable data transmission, lower latency, and more stable connection
    * Firmware: Reprap/Klipper
    * Driver support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Driver mode support: TMC: UART/SPI
    * Display: Serial touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Using repap firmware, it can use extended ESP32 WiFi module, supports 1.6m/s
    * Unique PWM Fan MOS board design, easy to replace when damaged, reduces customer repair difficulty
    * PCB uses 2OZ copper thickness, capable of supporting larger current
    * 3 ADC interfaces, supports up to three thermistors
    * 6 IO interfaces, can be used for leveling and limit switches
    * 4 heater interfaces, supports up to four heaters
