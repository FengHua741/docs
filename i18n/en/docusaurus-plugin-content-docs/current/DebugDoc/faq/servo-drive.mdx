# External Drive Usage Tutorial

## Precautions

:::danger Notice

* This tutorial is based on Klipper for control
* The module uses common anode wiring

:::


## Drive Wiring
    :::info

    * Only one of the following two methods needs to be selected

    :::
    * First wiring method
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/servo-drive.png').default} size="80%" align="center" />
    * Second wiring method
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/servo-drive-1.png').default} size="80%" align="center" />

## Drive Microstepping Jumper

    * No drive jumpering is required!!!

## Built-in DIP Switch on the Drive
    * It is recommended to use `3200` for the built-in DIP switch on the drive
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/3200.png').default} size="80%" align="center" />

## Klipper Reference Configuration

    * Disable the corresponding driver configuration, such as `[tmc5160 stepper_x]` or `[tmc2209 stepper_x]`
    * Set `microsteps:` to `16`
    * Add `step_pulse_duration: 0.000004`
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/cfg.png').default} size="50%" align="center" />

## Pulse Explanation for the Driver

    :::info

    * If the configuration is correct but the model prints with offset, modify `step_pulse_duration: 0.000004`
    * Increase this value `0.000004`, for example, to `0.000005` or `0.000006`, etc.

    :::
