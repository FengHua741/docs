# TMC Error Related

## `Unable to read tmc uart ‘stepper_x’ register IFCNT` or `Unable to read tmc spi ‘stepper_x’ register register`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart.png').default} size="80%" align="center" />

    2. Error Cause

        * Unable to communicate with the driver

    3. Main Solution Methods

        * Check if the UART pin configuration for the corresponding driver is correct
        * Check if the UART jumpers are correctly set
        * Ensure that the driver is powered
        * Replace the driver or reconfigure the driver port
        * If it's a driver from another manufacturer, you need to determine if the driver's UART pin is consistent with FLY's. The FLY driver's UART pin is the fourth pin from the top on the left side
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/2209uart.png').default} size="80%" align="center" />
        * If the driver's UART pin is on the fifth pin, you can solve it using the following jumper settings
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart1.png').default} size="80%" align="center" />

## `TMC XXXX reports error: GSTAT: 00000001 reset=1(Reset)`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/reset.png').default} size="80%" align="center" />

    2. Error Cause

        * Driver reset

    3. Main Solution Methods

        * Poor power connection
        * Motor disconnected
        * Connect the motor's metal casing with the motherboard or tool board's GND together

## `TMC 'extruder' reports error: DRV_STATUS: c00d0103 otpw=1(OvertempWarning!)`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/overtemperror.png').default} size="80%" align="center" />

    2. Error Cause

        * Driver overheating!
        * Not motor overheating!

    3. Main Solution Methods

        * Reduce operating current
        * Remove holding current
        * Check if the driver's sampling resistor is correct
        * Improve driver cooling, such as adding a fan

## `TMC ‘extruder’ reports error: DRV_STATUS: 601f1000 s2vsa=1(ShortToSupply_A!) csactual=31 ola=1(OpenLoad_A!) olb=1(OpenLoad_B!)`

    1. Error Details

        * No image available

    2. Error Cause

        * Incorrect driver current check

    3. Main Solution Methods

        * Check if the resistance values of the two sets of coils in the stepper motor are consistent
        * Check if the wires inside the stepper motor or components are loose or short-circuited
        * Turn off silent mode

## `TMC 'extruder' reports error: DRV_STATUS: 00190060 s2vsb=1(ShortToSupply_B!) ola=1(0penLoad_A!) cs_actual-25`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/ab.png').default} size="80%" align="center" />

    2. Error Cause

        * Wrong wiring or issue with the motor itself

    3. Main Solution Methods

        * Check if the resistance values of the two sets of coils in the stepper motor are consistent
        * Check your stepper motor and connections
        * Check the motor wiring sequence, whether one group is on the left and the other on the right
