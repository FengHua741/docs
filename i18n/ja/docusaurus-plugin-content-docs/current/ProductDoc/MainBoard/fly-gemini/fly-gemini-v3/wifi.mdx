# Wi-Fiの接続設定

## Wi-FiアンテナをGeminiに取り付けてください。

```
* 赤い矢印で示されている場所がWi-Fiアンテナの接続口です。
* Wi-Fiアンテナの接続口は`ipex1`タイプです。
* アンテナをGeminiに取り付けてください。取り付けない場合、信号が弱くなりネットワークに接続できない可能性があります。
```
