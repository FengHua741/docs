```
:::tip

*上位機が正常にインターネットに接続されていることを確認してください
*ファームウェアをコンパイルする前に、SSHを使用してネットワーク経由で上位機にログインしてください
*シリアルポートツールを使用して上位機にログインしないでください

:::
```

## Klipperファームウェアのコンパイル設定ページの操作手順

```
:::tip

キーボードの入力モードを半角モード、つまり英語モードに設定してください

:::

1.キーボードの上キー`↑`、下キー`↓`でカーソルを上下に移動してメニュー項目を選択します
2.確認キー`Enter`またはスペースキー`Space`でメニューを選択するか、サブメニューに入ります
3.退出キー`ESC`で一つ上のメニューに戻ります
4.`Q`キーでKlipperファームウェア設定ページを終了します
5.Klipperファームウェア設定ページを終了する際に、プロンプトが表示された場合は`Y`キーを押して設定ファイルを保存します
:::warning

設定ページのオプションが少ない場合は、まず`[ ] Enable extra low-level configuration options`を選択して、一部の隠しオプションを表示してください

:::
```
