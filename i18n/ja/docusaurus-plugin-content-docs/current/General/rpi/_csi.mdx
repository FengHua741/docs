## CSIの使用方法

```
* イメージの書き込みが完了したら、/boot/config.txtに移動し、`camera_auto_detect=1`を`camera_auto_detect=0`に変更して、カメラのモデルに応じた設定を追加してください。
* 以下は参考設定です。

```bash
dtoverlay=imx219,cam1
```
